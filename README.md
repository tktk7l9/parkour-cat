# 🐈 パルクール猫 — Parkour Cat

A Three.js / WebGL parkour game where a cute cat acrobatically dashes through city streets and nature.

**[▶ Play Now](https://tktk7l9.github.io/parkour-cat/)**

![Three.js](https://img.shields.io/badge/Three.js-r160-black?logo=three.js) ![WebGL](https://img.shields.io/badge/WebGL-2.0-red) ![License](https://img.shields.io/badge/license-MIT-blue)

## Controls

| Key | Action |
|-----|--------|
| `W` `A` `S` `D` | Move (camera-relative) |
| `Space` | Jump / Double Jump |
| `Shift` | Dash |

## Features

- **Cute cat model** — tabby stripes, big anime eyes, toe beans, white socks, multi-segment curling tail
- **Idle animations** — sit, groom, look around, tail flick, blink, stretch
- **24 parkour tricks** with HUD name display and combo system
- **5 fish varieties** — Koi, Mackerel, Salmon, Goldfish (rare), Rainbow Fish (ultra rare)
- **Mice AI** — wander the streets and flee when the cat approaches
- **3D world** — city buildings with windows/awnings/pipes, rooftop planks, park benches, lamp posts, trees, flowers, rocks, distant hills, animated clouds
- **Physics** — wall running, wall kicks, coyote time, double jump, auto parkour roll on heavy landing

## Run Locally

Just open `index.html` in a browser — no server or build step required.

```bash
open index.html
```

Requires a browser with WebGL 2.0 support (Chrome, Firefox, Safari, Edge).
